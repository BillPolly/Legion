# My Project

## Setup

### Backend
1. Navigate to the backend directory
2. Run `npm install`
3. Start the server with `node server.js`

### Frontend
1. Open the frontend/index.html file in a web browser

## Project Structure
- /backend
  - server.js (Express server)
  - package.json
- /frontend
  - index.html
  - styles.css
  - app.js
