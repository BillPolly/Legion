Overview

Security Features

The system implements defense-in-depth against prompt injection attacks:

  • Pattern Detection: Regex-based detection of common injection attempts (e.g., "ignore previous instructions")
  • Delimiter Wrapping: XML tags force LLMs to treat external content as data, not instructions
  • Prompt Hardening: System prompts explicitly instruct agents to ignore injections
  • Link Validation: Only validated, accessible URLs are processed
  • Content Sanitization: All web content is checked and wrapped before LLM processing
Zero ML Models: Security uses lightweight regex patterns (~1ms overhead) instead of heavyweight ML models, keeping the system fast and API-only.
